System Events Analysis - Help

This application analyzes system event traces from Android devices using eBPF tracing. It provides:

  • Event Timeline - Visual timeline of system events
  • Statistics - Device usage and event type analysis
  • Advanced Analytics - High-level behavior analysis with charts
  • Export - Download data in CSV or JSON format

Upload .trace files for analysis:

  1. Click "Choose File" and select your .trace file
  2. Click "Process Trace" to start processing
  3. Monitor the progress bar during processing
  4. Once complete, the interface will automatically reload with new data
Tip: Large trace files may take several minutes to process.

Use filters to focus your analysis:

  • PID Filter - Filter events by process ID
  • Device Filter - Filter events by device ID
  • Timeline Zoom - Zoom in/out on the timeline visualization
  • Export - Download filtered data in various formats
Export Options: Export events or analysis results as CSV or JSON.

Advanced analysis features for detailed insights:

Window Size
Number of events to analyze in each window (default: 1000)
Overlap
Number of events to overlap between windows (default: 200)
Target PID
Process ID to focus analysis on (auto-detected if empty)
Note: Advanced analytics may take time for large traces.
